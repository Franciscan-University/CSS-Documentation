Welcome to the CSS and web brand guide for Franciscan University. This guide is an ongoing project to aid web developers in proper use of Franciscan's CSS files.

The style guidelines and classes here should be followed in the spirit of the new 2024 branding. While it is good to keep styling from page to page as uniform as possible, there will be incidents that call for styles outside of these guidelines. This is where good knowledge of the new branding, along with clear communication with all involved in the project, will be essential.
