Rows

Jump to:

  • Common Mistakes
.row { display:-ms-flexbox; display:flex; -ms-flex-wrap:wrap; flex-wrap:wrap; width: 100%!important; }

The row class is useful for splitting up documents into different sections by reaching 100% of the width of the element it is contained in. So if it's used immediately in the body tag, where the body streches the entirety of the page, the element with the row class will be stretched the entirety of the page. In the example below, its parent element class is site-para-box, so the element with class row stretches to the full width of site-para-box.

In its use with the column classes, row has the display: flex style built in. It also allows content within it to wrap—which can be an advantage or a disadvantage. On one hand, if you have many columns and don't want to worry about starting a new row each time the column length exceeds 12, row already takes care of that for you. However, if you have other elements in row that you'd like to be stacked vertically, but those elements added do not exceed a width of 100%, they will align horizontally instead.

Content and Paragraph Boxes

There are two main ways of containing page content so that text and images aren't hitting the sides of the screen/viewport: site-content-box and site-para-box. The content box class (site-content-box) is slightly bigger, stretching to 85% of the parent element's width, whereas the paragraph box class (site-para-box) is only 75% of the parent's width.

Originally, only the paragraph box existed, and it is still the standard for sections that contain text-only or heavy-text content areas. A larger section was needed, however, for content that contained wider content, such as an array of professor profiles or an image heavy section. Thus, site-content-box was created.

Both collapse at 500px to 90% and 1em padding on either side.

.site-para-box { width: 75%; margin: auto; } .site-content-box { width: 85%; margin: auto; } @media only screen and (max-width: 500px) { .site-para-box, .site-content-box { width: 90%; padding-left: 1em; padding-right: 1em; } }

Adding Maximum Widths

Oftentimes, you will want to add a 'max-width' style to the site-content-box or site-para-box element in order to keep the content from stretching too wide. The two classes do not have a set maximum width in order to allow greater flexibility to the developer. Typical max-width settings are around 1300px, but different situations will call for different max-widths.
